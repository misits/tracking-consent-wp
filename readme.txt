The WP Tracking Consent Plugin adds Matomo tracking code to your WordPress site with user consent functionality. It provides an easy-to-use interface for managing consent settings and integrates seamlessly with Matomo Analytics.

== Description ==

The WP Tracking Consent Plugin provides a straightforward solution for integrating Matomo analytics tracking into your WordPress site, with a focus on user consent. The plugin facilitates easy management of tracking settings and user consent, ensuring compliance with privacy regulations. 

Key features include:
- User consent management for analytics tracking.
- Customizable consent form with Accept/Decline options.
- Seamless integration with Matomo Analytics.
- Admin interface for managing tracking details and consent form appearance.

== Installation ==

= From within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **WP Tracking Consent**.
3. Install and activate the plugin.

= Manual Installation =

1. Upload the `wp-tracking-consent` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the WP Tracking Consent plugin.

= After Activation =

1. Go to 'WP Tracking Consent' in the admin menu to configure the plugin.

== Frequently Asked Questions ==

= How do I find my Matomo Site ID and Auth Token? =

You can find these details in your Matomo dashboard under 'Settings'.

= Can I customize the appearance of the consent form? =

Yes, the plugin offers customizable options for the consent form including text, colors, and button styles.
